# Reproducibility Notes

For each important run, record the following.

- project repository commit hash
- `external/mujoco_wheeled_uav_simulator` submodule commit hash
- controller entry point
- simulator launch command
- parameter file used
- generated XML directory, typically `build/generated_xml`
- log output directory
- date and purpose of the run

This file is intentionally small. Expand it once the project begins producing real experiment results.
